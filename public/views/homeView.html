Secure Password Generator

Secure passwords should be as random and complex as possible, which makes them hard to remember. Easy to remember passwords often contain personal details or whole words that compromise their security. This application attempts to provide a fix.

This application takes sentences that the user should find easy to remember as it's input. Ideally, to make recollection easier, these sentences will be tied to the service the password is for. These sentences are run through a simple algorithm that takes the first letter of each word as characters for the password. These letters are then replaced by numbers which kinda look like the letter. Then, optionally, the next letter following a letter to number conversion is capitalized.
